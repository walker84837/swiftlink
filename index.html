Swiftlink

blazingly fast and memory-efficient

Easy to use, easy to configure.

Swiftlink is an open-source URL shortening service built in Rust using the Actix web framework. It is designed for high performance, security, and minimal memory overhead while offering full customizability.

How to use

For server admins

  1. Clone the repository from GitHub.
  2. Install Rust and compile the project.
  3. Configure your PostgreSQL database settings in the config file.
  4. Run the server and use our simple API to generate short links from long URLs.

For regular users

A client is coming soon, which makes shortening links easier. Documentation for the API is coming soon!

Features

  • Quick creation of short links from long URLs.
  • Customizable database support, including PostgreSQL.
  • High performance with low memory usage, thanks to Rust and Actix.
  • Configurable logging and error handling for easy troubleshooting.

Installation

Download the source code from our GitHub repository, install the necessary Rust dependencies, and compile the project. Update your configuration file with your PostgreSQL credentials, then launch the server. Detailed instructions are provided in the documentation.
