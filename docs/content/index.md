# Swiftlink documentation

Welcome to the Swiftlink documentation.

Swiftlink is a fast, self-hosted URL shortener written in Rust. This
documentation covers how to install, configure, and extend Swiftlink,
as well as how to use its API and CLI tools.

If you are new, start here:

- [Getting started](getting-started.md)
- [Server setup](server/setup.md)
- [API overview](api/index.md)
